OpenVPN
=======

Windows 10
----------

OpenVPN GUI > <usrname>.ovpn

WSL
---

1.	Can run through Windows 10 OpenVPN GUI
2.	If invoking listeners/reverse shells/etc on WSL;
	THEN disconnect Windows OpenVPN GUI and start `Linux`_ vpn.

Linux
-----

.. code-block:: Bash

	sudo openvpn <usrname>.ovpn
